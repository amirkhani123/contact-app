# Mini Contact Manager

یک **مینی اپلیکیشن مدیریت مخاطبین** ساخته شده با **React** و **Vite** برای مدیریت سریع و ساده‌ی مخاطبین.

--- 
مشاهده دمو [اینجا کلیک کنید ]( https://contact-k9fbb8dqi-amirkhani123s-projects.vercel.app/  ) 

--- 

## ویژگی‌ها

- افزودن مخاطبین جدید  
- ویرایش اطلاعات مخاطبین  
- حذف یک یا چند مخاطب همزمان  
- رابط کاربری ساده و واکنش‌گرا  
- استفاده از **React Icons** برای دکمه‌ها و تعاملات بصری  

---

## تکنولوژی‌ها

- **React 19**  
- **Vite** (Build Tool)  
- **React Icons**  
- **CSS Modules** برای استایل‌ها
